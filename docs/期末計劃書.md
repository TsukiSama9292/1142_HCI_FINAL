**觀眾情緒轉折與題材疲勞：分析「異世界」動畫敘事策略與觀眾留言情緒之序列模式**
組員：資工四B1143015林宣佑、資工四B1143021林成瑋、
資工四B1143009吳秉融、資工碩二R1343012鄭煒霖
<a id="_t0f0k2n7e0ur"></a>
# **一、摘要**
本企劃旨在探討日本動畫「異世界」題材的敘事策略與觀眾接受度。2010年代前期，多數異世界作品主打主角擁有外掛能力、後宮與無雙爽感；自2016年《Re:從零開始的異世界生活》（Re:Zero）將此題材轉向心理折磨與高壓敘事後，成功帶動了異世界題材的爆炸性主流化。本研究將擷取代表性異世界動畫片段與討論區留言，觀察敘事策略（如：外掛爽番 vs\. 心理折磨）如何影響觀眾留言情緒。期盼透過客觀的數據脈絡，釐清「何種異世界套路最能引發觀眾共鳴，或已導致審美疲勞？」
# **二、緒論**
動畫產業近年大量改編自「成為小說家吧」網站之作品，這類涵蓋轉生、外掛、慢生活等元素的故事，正式帶動了異世界題材的爆發。回顧發展脈絡，早期的《刀劍神域》與《遊戲人生》已率先將穿越與遊戲世界概念推向熱門。2016年則顛覆常規，以主角能力極弱、面臨死亡輪迴與嚴重心理壓力的「反套路」模式，將異世界從單純的爽番轉化為嚴肅黑暗的心理劇。此創新的極大成功，促使業界確認該類型的商業潛力，進而引爆2016至2018年的「異世界大時代」。
伴隨作品的大規模工業化生產，2017至2020年間市場開始浮現「異世界疲勞」現象。觀眾面對千篇一律的卡車轉生、作弊技能與超長標題，逐漸產生審美疲乏。據此，本案切入探討不同敘事策略在社群平台上的情緒互動效果，並檢視「演技與情節鋪陳」於現今網路文化中的實際影響力。
<a id="_agwfz2h7c9ap"></a>
# **三、研究目的**
本計畫透過延遲序列分析（Lag Sequential Analysis, LSA）方法，深入探討異世界動畫敘事策略如何影響觀眾留言的情緒反應與支持傾向 。研究聚焦以下三大面向：
1. 探討**不同題材**的**異世界動畫**的敘事表達類型與情節策略。
2. 分析觀眾留言中情緒走向與動畫敘事策略之間的關聯性。
3. 建構「敘事情境 → 觀眾情緒 → 討論風向」的序列模式，提供未來數位內容推薦系統與內容創作者作為衡量指標參考。
# **四、文獻探討**
**（一） 異世界題材之演變與受眾心理代償**
2010 年代後期，日本動畫產業大量改編自「小説家になろう（Narou）」網站之輕小說作品，使異世界（Isekai）題材快速成為動畫市場主流。此類作品多以「轉生」、「開掛」、「後宮」與「能力補償」作為核心敘事結構，提供觀眾對現實壓力之心理代償與逃避性滿足。研究指出，御宅族（Otaku）文化與異世界敘事之流行，與青年族群面臨社會競爭壓力、階級流動困難以及自我認同焦慮存在高度關聯\[1\]\[2\]。
《Re:Zero》則進一步改變異世界作品的敘事模式。其透過「死亡輪迴」、「創傷累積」與「心理崩潰」等反套路設計，使異世界敘事從傳統爽感滿足轉向高壓心理折磨與存在焦慮之情緒體驗。此種情緒反差亦使觀眾之觀看動機，從單純的代償性投射轉變為對角色痛苦與成長歷程之情感共鳴。
**（二） 動畫影像之敘事策略與召喚式結構**
動畫敘事並非單向資訊傳遞，而是透過角色塑造、鏡頭調度、情節安排與情緒轉折，建構觀眾之涉入機制。Iser 所提出之「召喚結構（Appellstruktur）」認為，文本會預先設計觀眾之情感參與路徑，使讀者於觀看過程中主動填補文本空缺並形成心理投射\[3\]。
動畫研究亦指出，情緒節奏與敘事張力之安排，將直接影響觀眾的沉浸感與情緒記憶\[4\]。據此，本研究將動畫內容切分為固定時間窗，並以角色情緒與敘事情境進行行為編碼，其目的即在於將動畫之召喚式結構轉化為可量化之敘事特徵矩陣，進一步分析不同敘事策略如何影響觀眾留言情緒之變化。
**（三） 網路社群討論與審美疲勞**
伴隨異世界動畫高度工業化與模板化生產，觀眾逐漸出現「異世界疲勞（Isekai Fatigue）」現象。當特定套路大量重複出現時，觀眾會由原先的沉浸式觀看，轉向帶有批判、吐槽與反諷性質之解構式觀看。此類現象在 YouTube、Niconico 與 Reddit 等社群平台中特別明顯\[5\]\[6\]。
網路留言研究指出，社群評論區不僅是情緒反應之紀錄空間，同時也是觀眾重新建構主體性與群體認同的重要場域\[7\]。觀眾透過吐槽、迷因（Meme）、情緒性評論與考據行為，重新詮釋作品中的公式化套路。據此，本研究透過 Python 擷取 YouTube 留言並進行情緒分類，能有效將非結構化文本轉換為可分析之受眾反應特徵。
**（四） 延遲序列分析於非結構化資料之應用**
由於動畫觀看與留言反應具有時間序列特性，傳統靜態統計方法難以有效分析「敘事情節」與「觀眾情緒」之動態轉移關係。Bakeman 與 Quera 指出，延遲序列分析（Lag Sequential Analysis, LSA）能透過轉移矩陣（Transition Matrix）、$Z$\-score 顯著性檢定與行為序列分析，辨識事件之間是否存在顯著的時序依賴關係\[8\]。
$$
Z=\frac{O-E}{\sqrt{E\left(1-p\right)}}
$$
LSA 已廣泛應用於教育科技、人機互動、行為科學與數位媒體研究之中\[8\]。本研究導入 LSA 方法，目的在於建立「動畫情節 → 觀眾情緒 → 社群風向」之序列模型，進一步分析特定敘事套路是否容易觸發觀眾之情緒反轉與審美疲勞。
# **五、研究方法**
本研究主要探討「動畫採用何種敘事策略與情節轉折」，以及上述策略是否影響觀眾的討論風向，據此將同步分析動畫片段與社群留言內容，藉以觀察兩者的對應關係。
本計畫以「每8秒為一單位」的切分方式，記錄影片的情節表現與角色情緒編碼，並於片段結尾標註該動畫的核心敘事策略。針對受眾反應，每部影片將擷取前50則熱門留言（排除無意義或空白內容），針對各留言標記情緒傾向。
影片編碼結果將產出Kappa值、Z\-score表與序列轉移圖等分析指標。後續會將各影片的序列轉移模式視為其敘事策略特徵，並與留言區的情緒分佈比例交叉比對，最終研判何種異世界敘事套路較能引發觀眾共鳴，抑或已導致受眾產生審美疲勞。
## 1.1 **資料來源與影片挑選標準**
本案預計挑選部具代表性的異世界動畫高潮片段進行比對（如：《Re:Zero》的崩潰橋段、《為美好的世界獻上祝福！》的搞笑解構橋段等）。挑選條件如下：
- 具備明確的異世界核心元素（如：外掛、轉生或死亡輪迴）。
- 影片或討論串具備高話題性，發生於關鍵情節轉折之後。
- 留言區活躍（至少有300則留言）。

## 1.4 **分析邏輯與方式（策略情緒 → 留言情緒）**
由於社群平台上的使用者留言屬於異質且具時間滯後性的非結構化資料，本研究雖無法直接建立個別留言與特定動畫秒數的絕對映射關係，唯實際執行時，得透過總體統計學與序列分析建立兩者之內在關聯。本研究將統整各部動畫片段的情緒序列組合，系統化分析其與討論區留言情緒分佈比例之共變關係，核心分析架構區分為以下四大步驟：
分析方式如下：
1. **情節序列與敘事特徵特徵化** 針對每部動畫片段，計算不同「情節轉折＋角色情緒（A至H）」之轉移矩陣（Transition Matrix）與發生頻率，定義出該片段之核心敘事策略（例如：持續性恐懼絕望、中立至正向決意之情緒過渡）。
2. **社群留言情緒分佈統計** 量化統計各影片討論區前50則熱門留言中，各類別社群情緒編碼（1至7）的百分比分佈，建構出該動畫片段所引發的受眾反應特徵向量。
3. **跨作品敘事效應對比分析** 交叉比對不同動畫作品間的轉移矩陣與留言情緒分佈。藉以辨識特定敘事特徵（例如：以《Re:Zero》為代表之「高頻率心理高壓與恐懼絕望」組合）與社群反應（例如：「沉浸共感」與「製作肯定」之高比例分佈）之對應法則。
4. **顯著性檢定與序列轉移可視化** 運用延遲序列分析（Lag Sequential Analysis, LSA）之客觀Z\-score檢定值，精準識別何種動畫情節序列具備統計顯著性，並藉此歸納出最能有效引導受眾產生共鳴，或導致「異世界審美疲勞（嘲諷吐槽）」之關鍵情節轉折路徑。
## 1.5 **工具與執行方式**
為確保研究數據之信效度與實務執行的嚴謹性，本計畫將導入結合人工審查與資料科學工具之混合式架構：
- **動畫片段觀測與行為編碼**
觀測團隊將使用行為編碼軟體，以每8秒為固定時框，針對影像畫面進行高精確度之角色情緒（字母編碼）逐段紀錄，並透過雙人獨立編碼計算 **Cohen's Kappa 值**，以維持評分者間信度。
- **社群留言自動化擷取與資料清洗**
利用 Python 撰寫客製化網頁爬蟲程式，自動化撈取討論區之文字資料，排除無意義之空白文字、重複刷榜與機器人留言。文本分類則採人工編碼結合語意標記（數字編碼），確保動漫流行語（如：開掛、胃痛、便當）之情緒判讀最佳化。
- **統計矩陣分析與進階視覺化**
運用統計軟體計算相依序列之 Z\-score 顯著性檢定。在資料視覺化層面，除了產出嚴謹的**序列轉移機率表**外，另將導入**桑基圖（Sankey Diagram）與狀態轉移圖（State Transition Diagram）**，將複雜的「動畫情節動態\-觀眾留言情緒」之質變歷程，進行高度可視化與結構化呈現。
