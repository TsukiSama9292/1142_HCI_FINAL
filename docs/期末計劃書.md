**從外掛爽番到心理折磨：分析「異世界」動畫敘事策略與觀眾留言情緒之序列模式**
組員：資工四B1143015林宣佑、資工四B1143021林成瑋、資工四B1143009吳秉融、資工碩二R1343012鄭煒霖

# **一、摘要**

本企劃旨在探討日本動畫「異世界」題材中，不同敘事策略如何影響觀眾的情緒反應與留言討論風向。2010年代前期，多數異世界作品主打主角擁有外掛能力、後宮與無雙爽感；自2016年《Re:從零開始的異世界生活》（Re:Zero）將其轉向心理折磨與高壓敘事後，成功帶動了異世界題材的爆炸性主流化。本研究將擷取代表性異世界動畫片段與YouTube留言，以延遲序列分析（Lag Sequential Analysis, LSA）為核心方法，觀察敘事策略（如：外掛爽番 vs. 心理折磨）如何影響觀眾留言情緒。研究導入心理學情緒分類理論（Ekman 基本情緒模型、Plutchik 情緒輪）與計算語言學之 GoEmotions 28 類細粒度情緒分類體系作為編碼架構之理論基礎，並參考近年自然語言處理（NLP）與小型語言模型（SLM）在社群媒體情緒分類之應用，建構一套整合影片角色情緒與留言者情緒的統一編碼框架，以期客觀釐清「何種異世界敘事套路最能引發觀眾共鳴」之序列模式。

# **二、緒論**

動畫產業近年大量改編自「成為小說家吧」網站之作品，這類涵蓋轉生、外掛、慢生活等元素的故事，正式帶動了異世界題材的爆發。回顧發展脈絡，早期的《刀劍神域》與《遊戲人生》已率先將穿越與遊戲世界概念推向熱門。2016年《Re:Zero》則顛覆常規，以主角能力極弱、面臨死亡輪迴與嚴重心理壓力的「反套路」模式，將異世界從單純的爽番轉化為嚴肅黑暗的心理劇。此創新極大成功，促使業界確認該類型的商業潛力，進而引爆2016至2018年的「異世界大時代」。

伴隨作品的大規模工業化生產，2017至2020年間市場開始出現關於「異世界疲勞」（Isekai Fatigue）的討論。觀眾面對千篇一律的卡車轉生、作弊技能與超長標題，逐漸浮現審美疲乏。然而，現有文獻多為現象描述與主觀評論，尚缺乏以量化序列分析檢驗「敘事策略—情緒反應」動態關係之實證研究。據此，本案切入探討不同敘事策略在社群平台上之情緒互動效果，藉由延遲序列分析與NLP情緒分類工具，系統化檢視異世界動畫的情節鋪陳如何驅動觀眾的情緒轉移。

# **三、研究目的**

本計畫透過延遲序列分析（Lag Sequential Analysis, LSA）方法，深入探討異世界動畫敘事策略如何影響觀眾留言的情緒反應與討論傾向。研究聚焦以下三大面向：

1. 建構一套**整合心理學情緒理論與NLP技術**之統一編碼框架，同時涵蓋影片角色情緒與留言者情緒，以利兩者之系統性比較。
2. 分析不同異世界動畫的**敘事策略特徵**（如：外掛爽感、心理折磨、搞笑解構）與**觀眾留言情緒分佈**之間的關聯性。
3. 建構「敘事情境 → 角色情緒序列 → 觀眾留言情緒」的序列轉移模型，提供未來數位內容推薦系統與內容創作者作為衡量指標參考。

# **四、文獻探討**

## （一）異世界題材之演變與受眾心理代償

異世界題材之敘事模式演變，反映了觀眾認知過程與審美偏好的轉向。根據 David Bordwell 的認知電影理論，影視作品的理解依賴於觀眾主動的情緒律動（Emotional Rhythm）與假設檢定過程 [2]。傳統異世界動畫多採取「外掛爽番」模式，其敘事策略傾向於提供目標明確、因果邏輯清晰的「古典好萊塢式」敘事結構 [2]。然而，隨著敘事模式的演進，出現了如心理折磨（Psychological Torture）等更複雜的「參數化」或「藝術電影」敘事傾向，強調角色的曖昧性與情緒張力 [2]。

Wolfgang Iser 的讀者反應理論進一步解釋了這種演變：文本中的「空白」（Gaps/Blanks）是驅動觀眾參與意義建構的核心動力 [1]。早期作品可能提供較少的認知空白，追求直接的情緒補償；而當代敘事策略則透過刻意創造情緒落差與預期挫折（Expectation Frustration），迫使觀眾在留言區進行更深層次的互動與意義填充 [1]。目前異世界動畫之相關文獻雖多集中於質性之社會學分析，但缺乏對「敘事策略如何系統性地誘發情緒反應」的量化序列研究 [2][3]。

## （二）情緒分類理論——從 Ekman 基本情緒到 GoEmotions 28 類細粒度情緒

情緒分類的精準度直接影響行為編碼的效度。Paul Ekman 提出的基本情緒理論定義了六項跨文化普遍的情緒：憤怒、厭惡、恐懼、快樂、悲傷與驚訝 [4]。儘管 Ekman 的體系在跨文化研究中具有重要地位，但在處理如社群留言等精細化（Fine-grained）語料時，其範疇顯得較為受限 [6]。

相較之下，Robert Plutchik 的**情緒輪模型（Wheel of Emotions）** 提供了更具結構性的分類框架。Plutchik 提出了八項基本情緒（分為四對極性，如快樂對悲傷、恐懼對憤怒），並區分了情緒的強度維度 [5]。此外，Plutchik 的模型解釋了複合情緒的生成機制（如快樂加信任形成愛），這對於捕捉動畫觀眾複雜的觀影情緒至關重要 [5]。

在計算語言學領域，**GoEmotions 資料集**為情緒分類提供了現代化的標籤體系，其包含 **28 種精細分類**，涵蓋 12 種正面情緒（如讚賞、有趣、喜悅、愛）、11 種負面情緒（如憤怒、恐懼、悲傷、厭惡）、4 種模糊情緒（如困惑、好奇、領悟、驚訝）以及 1 種中性情緒 [6]。研究顯示這些情緒在社交媒體（如 Reddit）中具有顯著的關聯性與聚類特徵 [6]。GoEmotions 的研究亦指出，精細標籤可映射至 Plutchik 或 Ekman 的粗粒度分組，以平衡分類精度與統計效能 [6]。本研究的影片內容與留言情緒編碼，即奠基於 GoEmotions 28 類細粒度情緒體系，並參考 Plutchik 情緒輪模型作為理論參照框架。

## （三）NLP 與小型語言模型（SLM）於社群媒體情緒分類之應用

自然語言處理（NLP）技術的演進，大幅提升了大規模分析留言情緒的可行性。從早期的特徵提取方法發展至 **Transformer 架構**，特別是 **BERT**（Bidirectional Encoder Representations from Transformers），透過雙向語境建模，解決了傳統模型無法捕捉複雜上下文的問題 [7]。後續的 **RoBERTa** 透過動態遮蔽（Dynamic Masking）與大規模數據微調，進一步優化了情緒識別的魯棒性 [8]。

然而，在計算資源受限或追求推理效率的環境中，**DistilBERT** 展現了顯著優勢。透過知識蒸餾（Knowledge Distillation）技術，DistilBERT 在保留 BERT 約 97% 性能的同時，減少了 40% 的參數並提升了 60% 的推理速度 [9]。此外，現代小型語言模型（SLM）的研究亦顯示，針對特定任務進行監督式微調（SFT），其效能通常優於複雜的提示工程（Prompt Engineering）[10]。

針對 YouTube 留言之情緒分析，El Azzouzy et al.（2025）比較 BERT、GPT、RoBERTa、T5 等 Transformer 模型在留言情感分類上的表現，發現 RoBERTa 在準確度與 F1 分數上均最佳 [11]。Visariya et al.（2025）則提出以 NLTK 為基礎之留言分析流程，可擴展至即時情緒監測應用 [12]。針對動漫評論等非正式語言，GoEmotions 預訓練模型展現了強大的泛化能力，能有效處理口語化與細微的情緒表達 [6]。本研究參考上述文獻，規劃以 **DistilBERT 經 GoEmotions 資料集微調**進行留言情緒初篩分類，平衡分類準確度與在地部署可行性。

## （四）動畫影像之敘事策略與召喚式結構

動畫敘事可視為一種動態的溝通契約。Wolfgang Iser 認為文本具備「**召喚結構**」（Appellstruktur），透過預設的反應邀請結構來引導讀者 [1]。在異世界動畫中，創作者透過影像與劇情的組合，設定了「隱含觀眾」（Implied Reader）的角色，引發觀眾進行期待與期待挫敗（Expectation formation/frustration）的心理循環 [1]。

Bordwell 則將此過程視為「敘述」（Narration）與「風格」（Style）的交互作用。觀眾透過影格中的視覺提示、剪輯節奏與音效，將表層的「敘述事」（Syuzhet）轉化為心理層面的「故事」（Fabula） [2]。這種轉化過程具備時間性（Temporal Process），涉及預測與回溯認知 [1]。敘事理論中的「負性」（Negativity）——即未表達的背景與否定——正是引發情感波動的關鍵力量 [1]。

將這些質性理論轉化為量化框架時，可將動畫中的敘事轉折（如角色死亡、反擊勝利）視為**前事行為（Antecedent Behaviors）**，將觀眾的情緒留言視為**目標行為（Target Behaviors）** [2][3]。據此，本研究將動畫內容切分為固定時間窗，並以角色情緒與敘事情境進行行為編碼，其目的即在於將動畫之召喚式結構轉化為可量化之敘事特徵矩陣。

## （五）延遲序列分析（LSA）於多模態行為序列之應用

為了科學化地驗證敘事策略與觀眾反應的因果連帶，本研究採用 Bakeman 與 Quera 發展的**延遲序列分析（Lag Sequential Analysis, LSA）**。LSA 是一種統計方法，用於分析時間序列中事件間的依賴關係，判斷目標行為在特定前事發生後出現的機率是否顯著高於隨機水平 [3]。

其核心組件包括：
1. **轉移矩陣（Transition Matrix）：** 記錄行為 A（前事）與行為 B（結果）之間在不同「延遲」（Lag）下的發生頻率 [3]。
2. **Z 值顯著性檢定（Z-Score Significance Test）：** 透過觀察頻率與期望頻率的差異，計算 Z 值。當 Z > 1.96 時，代表該行為序列在 0.05 水準下具有顯著的依賴關係 [3]。

$$
Z = \frac{O - E}{\sqrt{E(1-p)}}
$$

其中 O 為觀察次數、E 為期望次數、p 為基線機率。

3. **調整後殘差（Adjusted Residuals）：** 提供行為關聯強度與方向的指標 [3]。

LSA 已被廣泛應用於臨床心理學的情緒調節研究與媒體研究中的觀眾反應分析 [3]。Montague et al.（2011）將其應用於醫病互動中視線行為之序列分析，成功辨識互動中的主從引導模式 [13]。最新研究更將其擴展至情緒調節的親子互動序列（2025）[14]以及多模態教學分析 [15]。由於 LSA 能處理具備相互排斥性且隨時間演進的編碼範疇，因此極為適合分析「特定情節 → 角色情緒序列 → 觀眾即時反應」的動態轉移規律 [3]。本研究導入 LSA 方法，目的在於建立「動畫情節 → 觀眾情緒 → 社群風向」之序列模型，量化分析特定敘事套路是否系統性地觸發特定情緒轉移模式。

# **五、研究方法**

本研究主要探討「動畫採用何種敘事策略與情節轉折」，以及上述策略是否影響觀眾的討論風向，據此將同步分析動畫片段與社群留言內容，藉以觀察兩者的對應關係。

本計畫以「每8秒為一單位」的切分方式，記錄影片的情節表現與角色情緒編碼，並於片段結尾標註該動畫的核心敘事策略。針對受眾反應，每部影片將擷取前50則熱門留言（排除無意義或空白內容），並進行情緒分類。

## 1.1 資料來源與影片挑選標準

本案預計挑選6部具代表性的異世界動畫高潮片段進行比對，涵蓋三種主要敘事策略類型：

| 類型 | 敘事策略 | 代表作品 |
|------|---------|---------|
| 外掛爽感型 | 主角擁有壓倒性能力，戰鬥輾壓、後宮建構 | 《我想要成為影之強者》、《關於我轉生變成史萊姆這檔事》 |
| 心理折磨型 | 死亡輪迴、創傷累積、心理崩潰 | 《Re:Zero》、《來自深淵》 |
| 搞笑解構型 | 公式化套路之反諷、日常搞笑 | 《為美好世界獻上祝福！》、《打工吧！魔王大人》 |

挑選條件如下：
- 具備明確的異世界核心元素（如：外掛、轉生或死亡輪迴）。
- 影片或討論串具備高話題性，發生於關鍵情節轉折之後。
- 留言區活躍（至少有150則留言）。
- 三種敘事策略類型各選2部以上，以利跨類型比較。

| **V** | Remorse | 自責 | 負面 | 後悔、內疚、自責的表現 | 「都是我的錯…」、「主角太自責了」 |
| **W** | Sadness | 悲傷 | 負面 | 哭泣、悲傷、情緒低落 | 「看到哭」、「這集太催淚了」 |
| **X** | Confusion | 困惑 | 模糊 | 困惑表情、歪頭、不理解 | 「什麼？這是什麼展開？」、「看不懂」 |
| **Y** | Curiosity | 好奇 | 模糊 | 專注觀察、提問、探索 | 「這個世界觀好有趣」、「想知道更多設定」 |
| **Z** | Realization | 領悟 | 模糊 | 突然理解、恍然大悟的表情 | 「原來如此！」、「我懂了！這是伏筆」 |
| **@A** | Surprise | 驚訝 | 模糊 | 驚訝表情、瞪大眼睛、震驚 | 「什麼？居然是他？」、「這反轉太神了」 |
| **@B** | None | 無/不適用 | — | 分析角色不在畫面中 | 無意義留言／已排除 |
| **@C** | Neutral | 中性/無明顯情緒 | 中性 | 角色無明顯情緒表現、平靜狀態 | 「還行吧」、「普普通通」、「沒什麼感覺」 |

> 編碼採用字母序列表示：A–Z 對應前 26 類情緒，@A/@B/@C 分別對應 Surprise、None 與 Neutral。 

**此編碼框架之設計優勢**：
1. **理論基礎**：立基於 GoEmotions 28 類細粒度情緒分類體系 [6]，並可映射至 Plutchik 情緒輪或 Ekman 基本情緒之粗粒度分組，具備心理學與計算語言學之雙重信度基礎。
2. **雙向映射**：同一套編碼同時適用於影片角色情緒與留言情緒，便於進行跨模態比較。
3. **粒度對稱**：28 類情緒維度涵蓋正面、負面、模糊與中性情緒，足以區分常見動畫情緒反應，且可依分析需求進行聚合（如合併為正面/負面/模糊/中性四大類），在細粒度與統計檢定力之間取得平衡。

**編碼流程與信度**：觀測團隊將使用行為編碼軟體，以每8秒為固定時框進行四人獨立編碼，並計算 **Cohen's Kappa 值**以確保評分者間信度（目標 Kappa ≥ 0.60）。

## 1.3 NLP輔助留言情緒分類

為強化留言情緒分類之客觀性與可再現性，本研究導入小型語言模型（SLM）輔助分類流程：

1. **資料清洗**：排除空白、重複、機器人留言及非該影片語言之內容。
2. **留言擷取**：以 Python 撰寫 YouTube API 爬蟲，自動擷取影片前50則熱門留言。
3. **SLM初篩分類**：以 DistilBERT-base-uncased（經 GoEmotions 資料集微調）進行 28 類情緒之初次分類，輸出各類別信心分數 [6][9]。
4. **人工覆核校正**：對於 SLM 信心分數低於門檻值（0.70）之留言，由研究團隊成員進行人工編碼。
5. **編碼一致性檢定**：計算 SLM 分類與人工編碼之符合率，作為自動分類品質之監控指標。

## 1.4 分析邏輯與架構：影片情緒 → 留言情緒之關聯建構

為回應教師「影片分析與留言分析需有關聯性」之意見，本研究的核心分析路徑如下：

![架構圖](./分析邏輯與架構.png)

具體分析步驟如下：

### 步驟一：情節序列與敘事特徵特徵化
針對每部動畫片段，計算不同「角色情緒（28 類細粒度情緒）」之轉移矩陣與發生頻率，並可依分析需求將編碼聚合為正面（A、B、C、D、E、F、G、H、I、J、K、L）、負面（M、N、O、P、Q、R、S、T、U、V、W）、模糊（X、Y、Z、@A）與中性（@C）四大類進行統計。據此定義該片段之**核心敘事特徵向量**，例如：
- 「持續性恐懼絕望型」：S、T、U 等高頻出現，低情緒多樣性
- 「中立→正向決意型」：Z→J 或 W→J 的顯著轉移路徑
- 「高強度情緒擺盪型」：情緒類別間頻繁轉換，高 Shannon entropy

### 步驟二：社群留言情緒分佈統計
量化統計各影片討論區前50則熱門留言中，各類別社群情緒編碼（28 類細粒度情緒）的百分比分佈與情緒多樣性指標，並進行正面/負面/模糊/中性四大類之聚合統計，建構出該動畫片段所引發的**受眾反應特徵向量**。

### 步驟三：跨作品敘事效應對比分析
交叉比對不同敘事策略類型（外掛爽感/心理折磨/搞笑解構）的轉移矩陣與留言情緒分佈，辨識以下關鍵問題：
- 心理折磨型敘事是否系統性地引發較高的 **負面情緒（S、W、T、U）** 比例？
- 外掛爽感型是否關聯於較高的 **正面情緒（F、H、J、A）** 比例？
- 搞笑解構型是否觸發較高的 **厭惡／嘲諷類情緒（Q、N、P）** 比例？

### 步驟四：顯著性檢定與序列轉移可視化
運用 LSA 之 Z-score 檢定值，精準識別哪些情緒轉移路徑具備統計顯著性（Z > 1.96, p < 0.05）。最終成果將以**桑基圖（Sankey Diagram）** 呈現「敘事策略類型 → 角色情緒序列 → 留言情緒」之完整轉移路徑。

### 1.5 工具與執行方式

為確保研究數據之信效度與實務執行的嚴謹性，本計畫導入結合人工審查、NLP工具與統計軟體之混合式架構：

- **動畫片段觀測與行為編碼**：基於人工進行行為編碼，以每8秒為固定時框進行四人獨立編碼，計算 Cohen's Kappa 值維護評分者間信度。
- **社群留言自動化擷取與NLP分類**：利用 Python + YouTube API 擷取留言；以 DistilBERT（經 GoEmotions 微調）進行SLM初篩分類，人工校正信心不足之樣本。
- **統計矩陣分析與進階視覺化**：以 Python（Pandas）計算 Z-score 顯著性檢定；以 Plotly 產出序列轉移機率表與桑基圖。

# **六、預期成果與討論**

本研究預計產出以下成果：

1. **一套整合情緒分類框架**：奠基於 GoEmotions 28 類細粒度情緒分類體系 [6]，同時適用於影片角色情緒與留言情緒編碼，可供後續媒體情緒分析研究參考使用。
2. **多部異世界動畫之情緒序列轉移矩陣**：量化呈現不同敘事策略的情緒動態特徵，為「異世界敘事類型學」提供數據基礎。
3. **敘事策略—觀眾情緒對應法則**：透過跨作品比較，歸納「何種敘事情境容易觸發何種觀眾情緒」之經驗法則。
4. **NLP情緒分類之在地化驗證**：評估 DistilBERT 在動畫留言情緒分類上的表現，為SLM於中文／日文混合社群文本之應用提供實證數據。

本研究的限制包括：（1）僅分析熱門留言（前50則），可能遺漏長尾討論內容；（2）YouTube 留言本身具有平台特定性，推論至其他社群平台（如Reddit、Niconico）須謹慎；（3）SLM分類對動漫特定用語（如「胃痛」、「便當」、「開掛」）可能辨識度不足，需仰賴人工校正補強。
