# [GoEmotions - 28 種情緒根據情感傾向](https://huggingface.co/datasets/google-research-datasets/go_emotions)
1. 正面情緒 (Positive Emotions) — 共 12 種
- 讚賞 (Admiration)：對令人欽佩或值得尊敬的事物的感受
- 有趣 (Amusement)：覺得某事好笑或感到愉快
- 認可 (Approval)：持有或表達贊成的意見
- 關心 (Caring)：對他人表現出親切與關注
- 慾望 (Desire)：強烈的想要某物或希望某事發生的感受
- 興奮 (Excitement)：感到巨大的熱情與渴望
- 感激 (Gratitude)：感謝與賞識的感受
- 喜悅 (Joy)：愉快與幸福的感受
- 愛 (Love)：強烈的正向認可與情感連結
- 樂觀 (Optimism)：對未來或成功的希望與信心
- 自豪 (Pride)：因自身或相關人士的成就而感到滿足
- 寬慰 (Relief)：焦慮或痛苦釋放後的放鬆感
2. 負面情緒 (Negative Emotions) — 共 11 種
- 憤怒 (Anger)：強烈的不滿或敵對感
- 煩惱 (Annoyance)：輕微的憤怒或躁動感
- 失望 (Disappointment)：因期望落空而感到的悲傷或不滿
- 不認可 (Disapproval)：持有或表達負面、不贊成的意見
- 厭惡 (Disgust)：因不快或冒犯性事物引起的強烈反感
- 尷尬 (Embarrassment)：自覺、羞愧或窘迫的感受
- 恐懼 (Fear)：害怕或擔憂的感受
- 悲痛 (Grief)：強烈的哀傷，尤指因喪親引起
- 緊張 (Nervousness)：憂慮、擔心或焦慮感
- 自責 (Remorse)：後悔或內疚的感受
- 悲傷 (Sadness)：情感上的痛苦、悲哀
3. 模糊或中性情緒 (Ambiguous Emotions) — 共 4 種
- 困惑 (Confusion)：缺乏理解或感到不確定
- 好奇 (Curiosity)：強烈的求知慾或學習動力
- 領悟 (Realization)：意識到或覺察到某事
- 驚訝 (Surprise)：因出乎意料的事物感到震驚或吃驚
4. 中性狀態 (Neutral) — 共 1 種
- Neutral (中性/無明顯情緒)
